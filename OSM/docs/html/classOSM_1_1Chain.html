Detailed Description

template<typename _CoefficientType, int _ChainTypeFlag>
class OSM::Chain< _CoefficientType, _ChainTypeFlag >

Allow to create row/column sparse vectors.

The Chain class contains all algebraic functions that are related to vectors.

Template Parameters
_CoefficientTypeThe chain's coefficient types (default is OSM::ZCoefficient)
_ChainTypeFlagThe type of vector the chain is representing (default is OSM::COLUMN)
