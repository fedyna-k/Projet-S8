Detailed Description

template<typename _ChainType = OSM::Chain<OSM::ZCoefficient, OSM::COLUMN>>
class OSM::SparseMatrix< _ChainType >

Vector<Map> implementation of sparse matrices.

The SparseMatrix class contains all algebraic functions that are related to matrix.

Template Parameters
_ChainTypeThe matrix chains types (default is default OSM::Chain)
