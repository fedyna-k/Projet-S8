Detailed Description

template<typename _CoefficientType, int _ChainTypeFlag>
class OSM::SparseMatrix< _CoefficientType, _ChainTypeFlag >

Vector<Map> implementation of sparse matrices.

The SparseMatrix class contains all algebraic functions that are related to matrix.

Template Parameters
_CoefficientTypeThe chain's coefficient types (default is OSM::ZCoefficient)
_ChainTypeFlagThe type of vector the chain is representing (default is OSM::COLUMN)

Get a submatrix from the matrix.

Removes all indexes provided in the vector from the matrix and returns it.

Note
Will return a copy of the matrix if given vector is empty.
The submatrix will be rectangular.
